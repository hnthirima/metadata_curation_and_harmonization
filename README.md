# metadata_curation_and_harmonization
This repository contains learning material, code and documentation used for metadata curation and harmonization.

## A comprehensive R-based pipeline for harmonizing clinical metadata from multiple datasets into a standardized format for integrated analysis.
In this version, metadata from multiple melanoma RNA-Seq datasets were compiled and harmonized to generate one comprehensive file that may be used to integrate with molecular data. 

### Key Features
- ✅ Automated column mapping and standardization
- ✅ Unit conversion for time-based measurements
- ✅ Categorical value harmonization (200+ standardization rules)
- ✅ One-to-many column mapping support
- ✅ Built-in validation and testing framework
- ✅ Handles 1,100+ samples across multiple datasets
